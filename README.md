## Todo Routine App

Cross-platform mobile application for **task management and expense tracking** built with **Flutter**.
The application allows users to organize daily tasks, track expenses, and visualize spending statistics — all with **local data storage**, without requiring an internet connection.

## Task Management
- Create tasks with **start and end dates**
- View tasks in an **active tasks list**
- Mark tasks as **completed**
- Move tasks between **active and completed lists**
- Restore completed tasks
- Permanently delete tasks
- **Swipe gestures** for quick task management
- Task position is preserved when restored

## Expense Tracking
- Add expenses with:
  - amount
  - date
  - category
- View **expense statistics**
- Data visualization using **bar charts**
